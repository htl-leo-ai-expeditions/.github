# .github

This repository is the **organization profile home** for [HTL Leo AI Expeditions](https://github.com/htl-leo-ai-expeditions).

## Repository Structure

| Path | Purpose |
|------|---------|
| `profile/README.md` | Organization profile README displayed on the GitHub organization page |

## How It Works

GitHub automatically renders `profile/README.md` from this repository on the organization's public profile page at `https://github.com/htl-leo-ai-expeditions`.
